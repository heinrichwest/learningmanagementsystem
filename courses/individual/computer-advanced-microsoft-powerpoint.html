Advanced Microsoft PowerPoint

In today's corporate environment it is vital that your presentations are original, powerful and effective. In this course, we take you through the advanced features of PowerPoint to give you the tools to win your audience over.

Level: Advanced
Duration: 3 hours
Units: 11 units
Enroll Now Request Info

What You'll Learn

  • Create advanced graphics and SmartArt
  • Build interactive presentations
  • Use multimedia and animation effectively
  • Design professional templates and masters

Course Outline

Unit 1: Tables

+
  • Creating a table slide
  • Changing row height/column width
  • Inserting/deleting rows/columns
  • Applying borders and shading

Unit 2: Graphs

+
  • Creating graphs, i.e., column, pie, etc
  • Modifying and formatting graphs

Unit 3: Working with objects

+
  • Drawing and modifying lines/shapes/text boxes
  • Aligning objects
  • Rotating and flipping objects
  • Grouping and ungrouping objects

Unit 4: Using SmartArt

+
  • Inserting a SmartArt graphic
  • Converting an existing slide to SmartArt
  • Modifying the appearance of a SmartArt graphic

Unit 5: Templates and masters

+
  • Creating and modifying templates
  • Working with slide masters
  • Modifying backgrounds and colour schemes

Unit 6: Interactive slide show

+
  • Creating action buttons
  • Inserting hyperlinks

Unit 7: Multimedia automation

+
  • Adding animated pictures
  • Adding sounds
  • Adding videos/movies
  • Playing a CD

Unit 8: Importing and exporting

+
  • Importing and exporting between Word and PowerPoint
  • Importing and exporting between Excel and PowerPoint
  • Inserting slides from other presentations

Unit 9: Setting up a slide show

+
  • Rehearsing timings
  • Creating custom shows
  • Creating and broadcasting video for remote audiences

Unit 10: Slide setup options

+
  • Creating a self-running presentation
  • Packaging to a CD
  • Saving with embedded fonts
  • Saving a presentation as a show

Unit 11: Working with multiple presentations

+
  • Working with multiple presentations
  • How to copy between presentations
